- git init => inisialisasi folder github
- npm init => menjelaskan package yang dipakai di proyek kita
- remote dengan git remote add origin <nama-dari-repo-githubnya>
- git add . (prepare semua file yang terupdate untuk di upload ke github)
- git commit -m "penjelasan file yang di upload"
- git push -u origin "nama branch (main)"
- git checkout -b [nama brach baru] => membuat brach baru
- git pull [nama branch] => upload file di lokal
